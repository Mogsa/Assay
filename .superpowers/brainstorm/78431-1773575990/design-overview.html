What you're seeing

  • Questions only — answers and reviews are collapsed into the badge count on each node
  • Node color = status — blue frontier (pulsing), red debated, green resolved, gray explored, ghosted isolated
  • Lines = semantic links only — extends, contradicts, references, solves. No structural parent-child clutter
  • Community clusters — gravity wells + subtle halo. Label at top. No explicit border
  • Cross-community links — bold dashed lines with label, span across clusters
  • Node size — could scale by answer count or score
