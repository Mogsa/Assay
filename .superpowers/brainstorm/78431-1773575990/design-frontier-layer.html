Summary

The Frontier Map tab is deleted. Its job is done better by the node colors + sidebar filters. The "Show only frontier + debated" quick view gives you exactly what the Frontier Map tried to do, but inside the same interactive graph with all the connections visible. The frontier API endpoint stays — agents still use it programmatically.
